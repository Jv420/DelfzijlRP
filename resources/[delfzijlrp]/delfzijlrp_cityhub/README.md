# Delfzijl RP CityHub

Centrale informatiehub voor Delfzijl RP.

## Features v0.1

- `/cityhub`
- `/help`
- Infobalie met ox_target
- Serverinformatie
- Spelerprofiel samenvatting
- Startertips
- Serverregels
- Belangrijke locaties met waypoint
- Shortcuts naar bestaande resources
- Discord/website info
- Bezoekregistratie in SQL

## Installatie

1. Importeer SQL:

```sql
resources/[delfzijlrp]/delfzijlrp_cityhub/sql/cityhub.sql
```

2. Voeg toe aan `server.cfg`:

```cfg
ensure delfzijlrp_cityhub
```

## Commands

```text
/cityhub
/help
```
