# Delfzijl RP - Jumbo Premier

FiveM resource voor de gele Jumbo bezorgauto texture replace.

## Bestanden

Deze resource gebruikt de GTA/FiveM vehicle texturebestanden:

```txt
stream/premier.ytd
stream/premier+hi.ytd
```

Omdat dit binary `.ytd` bestanden zijn, moeten ze handmatig in de `stream` map gezet worden via GitHub Desktop, GitHub web upload of direct op de server.

## Installatie

Plaats de resource in je FiveM resources map:

```txt
resources/[vehicles]/delfzijlrp_jumbo_premier/
```

Zet daarna in `server.cfg`:

```cfg
ensure delfzijlrp_jumbo_premier
```

## Testen in-game

Spawn het voertuig als:

```txt
/car premier
```

Of via vMenu/adminmenu:

```txt
premier
```

## Let op

Deze mod vervangt de textures van de GTA Premier, omdat de bestanden `premier.ytd` en `premier+hi.ytd` heten. Voor een echte Caddy heb je een Caddy model/texture set nodig.
